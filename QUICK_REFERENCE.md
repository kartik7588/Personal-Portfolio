# Quick Reference

## Common Tasks

- **Run dev server**: 
pm run dev
- **Build for production**: 
pm run build
- **Preview build**: 
pm run preview

## Content Edits
- Projects/Skills/Experience: src/data/portfolio.ts
- Hero CTA/Resume: src/components/sections/Hero.tsx
- Navbar links: src/components/ui/Layout.tsx

## Troubleshooting
- Missing cards: check animations or hidden states in sections.
- 3D not showing: verify HeroScene.optimized.tsx import and WebGL support.
- Theme flash: confirm index.html theme script.
